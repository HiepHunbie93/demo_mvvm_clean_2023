# demo_mvvm_clean_2023

#### 1. Environment

- This tutorial run in MacOS M1 Ventura 13.2.1 (22D68)
- Homebrew 4.0.4
- NodeJS Version v18.11.0
- NPM 8.19.2
- LBRD 1.0 will run on Python 3.9
- LBRD 2.0 will run on Python v3.10.10
